# thecodingidiot.com

Companion repositories for [thecodingidiot.com](https://thecodingidiot.com) — the full C
learning journey, from terminal basics to Dreamcast 3D.

Each repo corresponds to one chapter and contains the tester and the full solution code.
Clone it, work through the chapter's implementation pages, then run the tester against your
own solution — or read the provided one.
